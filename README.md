# 3D Experiments

A collection of 3D rendering experiments built with vanilla JavaScript and Canvas.

## Experiments

| Experiment | Description | Demo |
|---|---|---|
| Rotating Particles | 3D particle system with rotation | [Open](experiments/rotating-particles/index.html) |

## Structure

```
3d-experiments/
├── index.html
├── src/
│   └── videos/
│       └── rotating_particles.webm
└── experiments/
    ...
```

## Running locally

Just open `index.html` in a browser — no build step required.

```bash
# Or serve with any server, example:
python3 -m http.server 3000
```
